Komodo's Contributor License Agreement (CLA)
============================================

This directory contains the CLA that we request of people who will
make significant contributions to Komodo. It is based on the Apache CLA:

http://www.apache.org/licenses/#clas
http://www.apache.org/licenses/icla.txt
